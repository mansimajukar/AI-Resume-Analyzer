# AI-Resume-Analyzer
AI-based resume analyzer that compares resumes with job descriptipns using NLP

##  Features
- Extracts resume text
- Compares resume with job description
- Calculates similarity score
- Shows matching percentage

## Technologies Used
- Python
- NLP
- Scikit-learn
- TF-IDF
- Cosine Similarity

 ## required libraries:
- pip install nltk spacy scikit-learn streamlit PyPDF2 docx2txt
- python -m spacy download en_core_web_sm

 ## How to Run:
- streamlit run app.py
